## Key properties

- Double-brushed microfiber for a smooth, soft hand feel.
- Moisture-wicking and temperature-regulating positioning for year-round use.
- Fitted sheets include all-around elastic.
- Mellanni's customer guide describes the microfiber sheets as hypoallergenic and suitable for sensitive skin.
- Sheet sets are described as fade, stain, shrink, pill, and wrinkle resistant.
- Machine washable: cold water, tumble dry low, no bleach, no ironing.
- OEKO-TEX Standard 100 certification is listed on the product pages.
- Free standard shipping is available for U.S. orders.
- Mellanni lists lifetime warranty and lifetime satisfaction support for its products.
- Official channels referenced across the docs include mellanni.com, Amazon, Walmart, eBay, and Wayfair.

## Start here

<Steps>
  <Step title="Measure the mattress height">
    Include any mattress topper, pillow-top layer, or mattress pad in the height measurement. Use the tallest point at the corner.
  </Step>
  <Step title="Choose the pocket depth">
    Use the standard Iconic sheet set for mattresses up to 16" deep. Use the extra-deep collection for mattresses up to 21" thick.
  </Step>
  <Step title="Pick the bundle">
    Choose a 4-piece set for a fitted sheet, flat sheet, and pillowcases. Choose a 6-piece set when you need four pillowcases. Choose a fitted sheet separate when only the bottom sheet needs replacement.
  </Step>
  <Step title="Confirm size and color">
    Size and color availability changes by product listing. Confirm the selected variant before checkout.
  </Step>
</Steps>
